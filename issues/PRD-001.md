# PRD: Podcast Transcription Feature

## Goal
Add a transcription option to the Podcast Summarizer to get the entire podcast transcribed and saved in a Markdown file.

## User Stories
- As a user, I want to see the full transcript of a podcast episode so I can refer back to specific details.
- As a user, I want the transcript to be saved as a Markdown file for easy reading and archiving.
- As a user, I want to be able to toggle transcription on/off to manage processing costs.

## Requirements
- Use Gemini's native audio transcription capabilities.
- Integrate the transcription step into the existing LangGraph workflow.
- Support batch processing for transcriptions.
- Export transcripts to a separate `transcriptions/` directory.
