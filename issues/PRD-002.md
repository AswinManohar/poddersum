# PRD: DeepEval Evaluation for Summarizing Agent

## Goal
Implement an automated evaluation framework to measure the quality, factual alignment, and accuracy of generated podcast summaries.

## User Stories
- As a developer, I want to know the quality score (0-1) of my summaries to detect regressions.
- As a developer, I want to catch "hallucinations" where the agent invents podcast details.
- As a developer, I want to compare different summary prompts to see which one performs better.

## Requirements
- Use `deepeval` for LLM-as-a-judge evaluation.
- Implement metrics for Summarization (Conciseness/Coverage) and Factual Alignment.
- Use the generated transcripts as "ground truth" for factual verification.
- Automate evaluation for batch-processed episodes.
